미국·한국 주식 포트폴리오를 무료로 관리하는 모든 방법을 안내합니다. Everything you need to know about managing your US & Korean stock portfolio — for free.

시작하기
Getting Started

📖 처음 사용하시나요? 📖 First Time Here?

별도의 설치 없이 브라우저에서 바로 사용할 수 있습니다. 모든 데이터는 이 기기 안에만 저장되며 외부 서버로 전송되지 않습니다. No installation needed — works directly in your browser. All data is stored locally on this device and never sent to any external server.

Step 1
Step 1

계정 만들기 Create Your Account

처음 접속하면 아래와 같이 계정 생성 화면이 나타납니다. 아이디와 비밀번호를 설정하면 암호화된 로컬 저장소에 안전하게 보관됩니다. When you first visit, you'll see the account creation screen. Set your ID and password — they are securely stored in an encrypted local vault.

▲ 계정 생성 화면 — 아이디(3자 이상), 비밀번호(6자 이상)를 입력하고 「계정 만들기」를 클릭하세요. ▲ Account creation screen — Enter an ID (3+ chars) and password (6+ chars), then click "Create Account".
  1. 1
    아이디 입력Enter ID — 3자 이상, 영문/숫자 권장3+ characters, alphanumeric recommended
  2. 2
    비밀번호 설정Set Password — 6자 이상, 잊어버리면 복구 불가 (복구 기능 없음)6+ characters. Cannot be recovered if forgotten (no recovery feature)
  3. 3
    「계정 만들기 →」 클릭 → 자동으로 로그인되어 앱이 시작됩니다. — Automatically logs you in and starts the app.
⚠️ 중요Important — 비밀번호는 PBKDF2-SHA256 (10만 회 반복)으로 해시 처리되어 저장됩니다. 분실 시 복구가 불가능하므로 안전한 곳에 보관해 주세요.Your password is hashed with PBKDF2-SHA256 (100,000 iterations). There is no recovery option — keep your password somewhere safe.
Step 2
Step 2

로그인 / 로그아웃 Login / Logout

계정 생성 후 재방문하면 로그인 화면이 표시됩니다. 세션은 브라우저를 닫아도 유지됩니다. On subsequent visits you'll see the login screen. Your session persists even after closing the browser.

▲ 로그인 화면 — 우측 상단의 「⎋ 로그아웃」 버튼을 누르면 로그인 화면으로 돌아옵니다. ▲ Login screen — Click "⎋ Logout" in the top-right to return here from the app.

핵심 기능
Core Feature

📋 포지션 추가 · 수정 · 삭제 📋 Add · Edit · Delete Positions

「+ 포지션 추가」 버튼을 클릭하면 종목 검색 모달이 열립니다. 티커를 입력하면 자동완성 목록에서 선택할 수 있습니다. Click "+ Add Position" to open the position dialog. Type a ticker to see autocomplete suggestions.

▲ 포지션 추가 모달▲ Add Position modal
▲ 티커 자동완성 — 섹터·비중 미리보기 제공▲ Ticker autocomplete with sector & weight preview
  1. 1
    티커 입력 — AAPL, SCHD, 005930.KS 형태로 입력. 한국 주식은 .KS (코스피) 또는 .KQ (코스닥) 접미사를 붙입니다.Enter ticker — e.g. AAPL, SCHD, 005930.KS. Korean stocks use .KS (KOSPI) or .KQ (KOSDAQ) suffix.
  2. 2
    수량 입력 — 보유 주수를 입력합니다.Enter shares — the number of shares you hold.
  3. 3
    평균 매수가 입력 — USD 또는 KRW 단위로 입력. 한국 주식은 원화(₩) 기준.Average buy price — in USD or KRW. Korean stocks use Korean Won (₩).
  4. 4
    배당수익률 입력 (선택) — 연간 배당수익률 %. 입력하면 배당 캘린더·리포트에 반영됩니다.Dividend yield (optional) — Annual yield %. Used in Dividend Calendar & Report.
  5. 5
    계좌 선택 — Brokerage, Roth IRA, 401k 등 원하는 계좌에 배정합니다.Select account — Assign to Brokerage, Roth IRA, 401k, etc.
▲ Positions 탭 — 종목 행을 클릭하면 수정, 🗑 아이콘으로 삭제(확인 팝업 있음), 헤더 클릭으로 정렬. ▲ Positions tab — click a row to edit, 🗑 to delete (with confirmation), click headers to sort.
▲ 필터 pill — All / US / Korea / ETF로 빠르게 필터링. ⚙ 버튼으로 커스텀 필터 추가 가능. ▲ Filter pills — quickly filter by All / US / Korea / ETF. Click ⚙ to add custom filters.
분석 탭
Analysis Tab

📊 Overview 탭 — 포트폴리오 한눈에 보기 📊 Overview Tab — Portfolio at a Glance

Overview 탭에서는 포트폴리오 총 평가액, 손익, 배당 통계와 함께 자산 배분·섹터 분석 차트를 확인할 수 있습니다. The Overview tab shows your total portfolio value, P&L, dividend stats, and allocation / sector analysis charts.

▲ Hero 섹션 — 총 평가액(USD), 평가손익, 투자 대비 배당수익률, 시장가 대비 배당수익률을 표시합니다. ▲ Hero section — shows total value (USD), unrealized P&L, yield on cost, and yield on market value.
▲ 자산 구성 도넛 차트 (자산/배당 탭 전환 가능) + 섹터 분석 가로 막대 차트. ▲ Asset allocation donut chart (switch between Asset/Dividend tabs) + sector horizontal bar chart.
▲ 포트폴리오 가치 추이 — 📸 스냅샷 버튼으로 저장한 기록이 꺾은선 그래프로 표시됩니다. 하루에 한 번 저장 가능. ▲ Portfolio value history — saves made with the 📸 Snapshot button appear as a line chart. One save per day.

📅 배당 캘린더 — 24개월 예상 배당 📅 Dividend Calendar — 24-Month Dividend Forecast

현재 보유 종목의 배당수익률을 기반으로 향후 24개월 예상 월 배당금을 종목별 색상으로 스택 바 차트에 표시합니다. Shows estimated monthly dividends for the next 24 months, stacked by stock and color-coded by yield tier.

▲ 배당 캘린더 — 현재 월은 흰 테두리로 표시. 각 막대에 마우스를 올리면 종목별 월 배당금을 확인할 수 있습니다. ▲ Dividend Calendar — current month has a white outline. Hover over each bar to see per-stock monthly dividend.

▲ 미래 예측 화면 — 슬라이더로 배당인상률·주가상승률을 조절하면 카드와 차트가 실시간 업데이트됩니다. ▲ Forecast screen — adjust sliders and cards + chart update in real time.
